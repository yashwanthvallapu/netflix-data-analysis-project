# netflix-data-analysis-project
Netflix Data Analysis Using SQL and Power BI
Netflix Data Analysis Project

Project Overview

This project analyzes Netflix content to understand trends in movies and TV shows using SQL and Power BI.

The goal of the project is to explore content distribution, ratings, genres, and country-wise production.

---

Tools Used

- SQL
- Excel
- Power BI

---

Dataset

The dataset contains information about Netflix movies and TV shows including:

- Title
- Director
- Cast
- Country
- Release Year
- Rating
- Genre
- Duration

---

Data Cleaning

Data cleaning was performed using Excel and SQL:

- Removed duplicates
- Handled missing values
- Split duration column into duration_type and duration_number
- Converted date_added into proper date format

---

SQL Analysis

Some business questions explored:

- Distribution of Movies vs TV Shows
- Top countries producing Netflix content
- Content added by year
- Most common content ratings
- Most popular genres

---

Dashboard

A Power BI dashboard was created to visualize the insights.

Key Visualizations

- Movies vs TV Shows distribution
- Content added by year
- Top 10 countries producing content
- Content rating distribution
- Most common genres

---

Key Insights

- Movies dominate the Netflix platform.
- The United States produces the most Netflix titles.
- TV-MA is the most common content rating.
- Action and Adventure is one of the most popular genres.
